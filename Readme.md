# .Net Core By Patika.dev

The repository contains .Net Core education set exercises.

* Http Protocol
* Restful API
* Controller, Route, Action Method
* Swagger Usage
* Sql, NoSql
* ORM
* Entity, Model, Mapper
* Fluent Validation
* Middleware
* Dependency Injection
* TDD (Test Driven Development)
* Authorization (Token based)
